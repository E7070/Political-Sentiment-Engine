# Political Sentiment Engine

Political Sentiment Engine is an AI-powered analytics platform designed to analyze political discussions from social media and transform unstructured text into actionable insights.

The project combines Natural Language Processing (NLP), Machine Learning, and Graph Neural Networks (GNNs) to classify sentiment, identify influential communities, detect emerging trends, and visualize public opinion through an interactive dashboard.

## Key Features

- Real-time political sentiment analysis
- NLP-based text preprocessing and classification
- Community detection using Graph Neural Networks
- Influencer and interaction network analysis
- Interactive dashboard with visual insights
- Trend monitoring and reporting

## Tech Stack

Python • Pandas • NumPy • Scikit-Learn • NLTK • spaCy • TextBlob • VADER • NetworkX • PyTorch • PyTorch Geometric • Flask • Streamlit
